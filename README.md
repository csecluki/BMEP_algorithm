# BMEP_algorithm
Implementation of BMEP (Bellman, Moore, d’Escopo, Papugo) algorithm used to solve shortest path problem.\newline
main.py returns full answer to shortest path problems using "manual" way, so it could be helpful for students struggling or learning this algorithm.
To get result to your task just run main.py script with path to your .txt file, which contains "description" of directed graph (flag -f).
